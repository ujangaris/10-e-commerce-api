            },

            "engines": {

                rm -rf .git - mac,
                git init 
                git add .
            pada .gitignore :
                /node_modules
                .env

                git commit -m "firt commit"
                heroku login

                    git remote -v

            pilih seting, klik Config Vars lalu semua source yang berada pada .env di isi satu persatu.

            pada terminal :
                git push heroku master

                
            pada browser untuk melihat hasil bisa klik Open app
